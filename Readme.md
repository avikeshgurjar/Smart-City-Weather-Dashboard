# Smart City Weather Dashboard 🌤️

Hey there! 👋 This is my first project using React. I built a simple,yet handy Weather Dashboard that lets you quickly check the current temperature for any city in the world. 

It was a fun little project to get my hands dirty with React fundamentals and fetching data from an external API!

## 🚀 What it does
- Type a city name and  fetches the current temperature in Celsius.
- Uses real-time data from the OpenWeatherMap API.

## 🛠️ Tech Stack
- **HTML,CSS,JAVAScript**
- **Reacy** 
- **OpenWeatherMap API**




### the API link:
- https://openweathermap.org/current

### features:
- background color change on the basis of temperature.
- a advice shown on the ui on the basis of weather condition.
